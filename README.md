# 💸 Expense Tracker (Flask Project)

## 📌 Overview

This is a **full-stack Expense Tracker web application** built using **Python Flask, HTML, CSS, and JavaScript**.
It allows users to manage their daily expenses, track spending, and maintain financial records in a simple and user-friendly interface.

---

## 🚀 Features

* 🔐 User Authentication (Register & Login)
* ➕ Add Expenses (amount + category)
* ❌ Delete Expenses
* 💰 Automatic Total Calculation
* 📊 Dashboard View
* 💾 Data stored using SQLite Database
* 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **Database:** SQLite

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone <your-repo-link>
cd expense-tracker
```

### 2️⃣ Install dependencies

```
pip install flask
```

### 3️⃣ Initialize Database

```
python init_db.py
```

### 4️⃣ Run the application

```
python app.py
```

### 5️⃣ Open in browser

```
http://127.0.0.1:5000
```
